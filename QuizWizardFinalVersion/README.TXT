PROJECT TITLE: QuizWizard
PURPOSE OF PROJECT: An exercise to practice with java, UML, and HTTLP
VERSION or DATE: 30/05/22
HOW TO START THIS PROJECT:
AUTHORS:  Daniele Del Piano
USER INSTRUCTIONS:
The project has 4 classes : Question, Quiz, QuizMaker, InputReader.
The QuizMaker will enable you to create a quiz with questions and a set number of answers set by
the user.
The QuizMaker will interactively allow users to:
• Create a new quiz and populate it with questions. Questions can be added, removed
or replaced based on their unique code.
• Print the quiz.
• Return the quiz tag (for internal use only);
